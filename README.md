## 项目概述

本项目实现了一个支持多架无人机同时起飞、协同飞行和智能避障的系统。系统能够根据无人机传感器的实时状态（紧急/警告/正常）进行动态避障决策。

## 核心功能

### 🚀 一键起飞
- 支持多架无人机同时起飞
- 通过 Shell 脚本快速启动整个无人机集群

### 🛡️ 智能状态监测与避障
系统持续监测各无人机状态，并基于以下等级进行避障决策：

| 状态等级 | 触发条件 | 避障策略 |
|---------|----------|----------|
| **EMERGENCY** | 即将碰撞、传感器严重异常 | 立即紧急避障或悬停 |
| **WARNING** | 检测到障碍物、距离较近 | 主动调整航向和速度 |
| **NORMAL** | 飞行环境安全 | 按预定航线正常飞行 |
